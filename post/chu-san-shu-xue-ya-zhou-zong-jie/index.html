初三数学压轴总结

2020-03-10

T25、26均考察以下内容：
纯代数、纯几何、代数几何综合（常在坐标系中考察）、新定义。

新定义题一般不会涉及过难内容，可归结于前三类。

在此记录一下做压轴题的经验。但是，适量的刷题仍然是最好的数学训练、提升方法。

纯代数

代数变形是比较有技巧和套路的，常见的方法有主元法、整体法、暴力法等。
现在所有的代数问题都可归结于方程问题或不等式问题，但要注意取值范围。
一定要注意的是，看见分式或二次根式就需要想到取值范围问题！

因式分解

常用方法有公式法、整体法、主元法、分组法、十字（长十字）法、因式定理、增项法等。
十分重要，是因为解方程与因式分解有直接关系。

此处仅涉及因式定理。
因式定理 = 蒙解\color{red}{\text{因式定理 = 蒙解}}因式定理 = 蒙解
什么意思呢？比如解方程 x4+3x3−24x2+27x−7=0x^4 + 3x^3 - 24x^2 + 27x - 7 = 0x4+3x3−24x2+27x−7=0，怎么办？
经过观察，发现 x=1x = 1x=1 是方程的一个解，则由因式定理可知 x4+3x3−24x2+27x−7x^4 + 3x^3 - 24x^2 + 27x - 7x4+3x3−24x2+27x−7 一定可以因式分解为 (x−1)⋅(…)(x-1)\cdot(\ldots)(x−1)⋅(…) 的形式，用大除法即可完成因式分解。
得 x4+3x3−24x2+27x−7=(x−1)(x3+4x2−20x+7)x^4 + 3x^3 - 24x^2 + 27x - 7 = (x - 1)(x^3 + 4x^2 - 20x + 7)x4+3x3−24x2+27x−7=(x−1)(x3+4x2−20x+7)
继续分解 (x3+4x2−20x+7)=0(x^3 + 4x^2 - 20x + 7) = 0(x3+4x2−20x+7)=0，发现 x=−7x = -7x=−7 是其一个解，则一定可以因式分解为 (x+7)⋅(…)(x+7)\cdot(\ldots)(x+7)⋅(…) 的形式。
